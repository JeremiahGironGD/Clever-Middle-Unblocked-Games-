About Clever-Middle

Clever-Middle is a simple Unblocked Games site that aims to allow users to bypass filters and access blocked games, we are working many methods, some of which prove to be unsuccessful.

We are currently working on a method to bypass filters using falsified iframes, we have had some success with this method, but it is not yet fully implemented.

We are also exploring other methods to achieve the same goal.

We use a token system to save data and connect to people's Clever accounts, WE DO NOT HAVE ACCESS TO ANY SENSITIVE DATA, we only have access to the user's name and login token.

We currently do not have data leak prevention measures in place, so please know that by linking your Clever account you are putting yourself at risk of having your token compromised.

Having your token will not effect you in any way, this is because tokens are one-time and will expire after use meaning that the token will be invalid after it has been used.

Expect minimal functionality at this time, as we are in testing stages(pre-alpha stages)

Check updates at https://github.com/Clever-Middle/clever-middle and you can also submit issues and feature requests here.

We are in early development, so expect bugs and issues, if you find any please report them to us.
